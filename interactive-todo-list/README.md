# Interactive To-Do List App

**Prepared by:** Sowmya Nagarajan  
**Assignment #4**

## Overview
Simple, responsive To-Do list app built with HTML, CSS and JavaScript. Features:
- Add tasks (with Enter or Add button)
- Toggle complete by clicking the task or checkbox
- Delete individual tasks
- Clear all tasks
- Live task counter ("X tasks remaining")
- Persistence using `localStorage`
- Double-click a task to edit (stretch goal)

## Files
- `index.html` — markup
- `style.css` — responsive styles (mobile-first)
- `script.js` — all functionality (DOM, events, storage)

## How to run
1. Unzip the project.
2. Open `index.html` in your browser.
3. Start adding tasks.

## Notes
- Tasks are saved to your browser's `localStorage` under key `todo_tasks_v1`.
- This is a front-end only project (no server required).
